什么是ANGULAR?

怎么用ANGULAR?

安装

1.先检查本机的npm

使用

npm 为你和你的团队打开了连接整个 JavaScript 天才世界的一扇大门。它是世界上最大的软件注册表，每星期大约有 30 亿次的下载量，包含超过 600000 个 *包（package）* （即，代码模块）。

2.使用npm全局安装Angluar

npm install -g @angular/cli

使用

创建工作空间

1.运行 CLI 命令 ng new，并提供一个名字 。

例如 ng new my-app

ng new 会提示你要把哪些特性包含在初始的应用项目中。

按 Enter 键接受默认值。

2.Angular CLI 会安装必要的 Angular npm 包及其它依赖。这可能要花几分钟。

等一会就好了。

还将创建下列工作空间和初始项目文件：

  • 一个新的工作空间，根目录名叫 my-app
  • 一个初始的骨架应用项目，也叫 my-app（但位于 src 子目录下）
  • 一个端到端测试项目（位于 e2e 子目录下）
  • 相关的配置文件

启动服务器

  1. 进入工作空间目录（my-app）。

    即cd my-app

  2. 使用 CLI 命令 ng serve 启动开发服务器，并带上 --open 选项。

即 ng serve –open.就可以在浏览器中打开可见

angular 默认的端口是4200；hexo 默认的端口是4000.

然后就可以在本地访问了。
